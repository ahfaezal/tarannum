# Tarannum.ai Glossary

Version: 1.0  
Date: 2026-06-26  
Last Updated: 2026-06-26

## Reference Audio

The authoritative qari audio used as the model for practice or analysis.

## Reference Timeline

The official time axis derived from the Reference Audio.

## Reference Playhead

The current playback position on the Reference Timeline. In Practice Live, it guides the student through the qari recitation.

## Reference Pitch Curve

The qari pitch contour plotted according to the Reference Timeline.

## Student Pitch Overlay

The student's detected pitch plotted over the Reference Pitch Curve. During Practice Live, it uses the Reference Timeline.

## Practice Live

The active training mode where the student follows the qari in real time.

## Recording Analysis

The review and scoring mode where a completed student recording is compared with the reference.

## Follow Zone

The region of the viewport where the Reference Playhead becomes visually centered and the graph moves beneath it.

## Viewport

The visible time window shown by the rendering engine.

## Auto Follow

The behaviour that moves the viewport to follow the Reference Playhead.

## Master Clock

The single timing source used by a rendering mode. In Practice Live, the Master Clock is derived from Reference Audio.

## Rendering Engine

The system responsible for drawing the Reference Timeline, Reference Playhead, Reference Pitch Curve, Student Pitch Overlay, and viewport state.

## Single Source of Truth

The authoritative source used to resolve timing, rendering, and behavioural decisions. In Practice Live, Reference Audio is the Single Source of Truth.
